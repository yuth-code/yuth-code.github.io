Edu-Tech DRM Architecture

Designed and audited the security infrastructure for a major learning platform. Prevented video piracy and unauthorized screen recording on rooted Android devices.

  • ✅ Implemented Widevine DRM (L1/L3 fallback strategy)
  • ✅ Blocked Screen Recorders via Overlay & FLAG_SECURE
  • ✅ Encrypted Local Storage with SQLCipher + key rotation

Approach: Conducted threat modeling across content delivery pipeline — identified 3 critical exfiltration vectors via rooted devices using custom Frida scripts to intercept DRM key exchange.

Outcome: Reduced piracy-related content complaints by ~80% post-deployment. Presented findings internally; solution now ships to all Android clients.
